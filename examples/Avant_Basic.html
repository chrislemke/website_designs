Typography

Rounded, Organic, Joyful

Playful, rounded, and organic lettering that avoids rigid or corporate typefaces. Bold and extra-bold for headings, regular or medium for body text. Generous sizing with confident, large headings and comfortable line spacing.

Avant Basic embraces maximalism as a deliberate contrast to minimalism. The term was coined in December 2020 by Emma Hope Allwood, describing it as "extravagance in the age of mechanical reproduction." Its visual language draws on pastel colors, retro-psychedelic patterns, and a combination of clean and curved lines -- popularized through Instagram and TikTok as an aesthetic that values playfulness, kitsch, and creativity over corporate restraint.

Visual Characteristics

Shapes, Forms & Effects

Curved and rounded elements are central. Everything is softened -- generous border-radius values, pill shapes, blob forms, wavy dividers, and organic geometry that feels hand-drawn but structured.

Layout

Maximalist but Organized

Lots of visual elements arranged with intentional rhythm. Neutral backgrounds as canvas, mixed scales, asymmetric balance, gallery-wall compositions, and layered depth.

CSS Techniques

Building the Aesthetic in Code

From generous border-radius values to organic blob shapes, checkerboard patterns to pastel-tinted shadows -- here are the key CSS techniques that bring Avant Basic to life on the web.

Border Radius & Soft Shapes
/* Generous rounding on all elements */
.card {
  border-radius: 24px;
}
.button {
  border-radius: 999px; /* pill */
}
.container {
  border-radius: 16px;
}
Checkerboard Pattern
.checkerboard {
  background-image:
    linear-gradient(45deg,
      #F4A7BB 25%, transparent 25%),
    linear-gradient(-45deg,
      #F4A7BB 25%, transparent 25%),
    linear-gradient(45deg,
      transparent 75%, #F4A7BB 75%),
    linear-gradient(-45deg,
      transparent 75%, #F4A7BB 75%);
  background-size: 40px 40px;
}
Blob / Organic Shapes
.blob {
  border-radius:
    60% 40% 50% 70% /
    50% 60% 40% 60%;
  /* Irregular border-radius creates
     organic blob forms */
}
Soft Shadows & Glows
.card {
  box-shadow:
    0 8px 32px
    rgba(244, 167, 187, 0.2);
}
.highlight {
  box-shadow:
    0 0 20px
    rgba(168, 216, 234, 0.4);
}
Pastel Gradients
.gradient-bg {
  background:
    linear-gradient(135deg,
      #A8D8EA 0%,
      #F4A7BB 50%,
      #FFCBA4 100%);
}
Wavy Section Edges
.wavy-divider {
  clip-path: polygon(
    0% 0%, 100% 0%, 100% 85%,
    75% 100%, 50% 85%,
    25% 100%, 0% 85%
  );
}
Influences

Standing on the Shoulders of Bold Design

Memphis Design
1960s & 70s Retro
Psychedelia
Mid-Century Modern

Memphis Design provides the primary visual ancestry -- bold geometry, clashing patterns, and bright pastels. The 1960s and 70s contribute retro color palettes, psychedelic swirls, and organic forms. Psychedelia brings flowing, melting shapes with saturated but dreamy color relationships. Mid-Century Modern supplies the clean structure underlying the maximalist surface layer.
