Duotone Techniques

Effects & Visual Treatments

From CSS blend modes to SVG filters, these techniques transform ordinary imagery into bold, two-color graphic statements. Hover to reveal the original.

CSS Blend Mode + SVG Filter Duotone
Indigo + Coral
Primary pair. CSS mix-blend-mode: color on a grayscale base creates the classic duotone mapping.
Navy + Magenta
Deep dark pole with vivid magenta highlights. High energy, contemporary, music-culture coded.
Purple + Gold
Warm, luxurious pairing. Royal purple shadows with sunlit gold highlights create rich depth.
Blend Mode Techniques
mix-blend-mode: color
The core duotone technique. A gradient overlay in 'color' blend mode on a grayscale image maps the tonal range to two colors.
mix-blend-mode: multiply
Single-color tint via multiply. Darkens the image while introducing the highlight color. Simpler but less true duotone.
mix-blend-mode: screen
Screen blend with the shadow color. Lightens the dark regions and tints them with the shadow hue. Ethereal, washed-out quality.
