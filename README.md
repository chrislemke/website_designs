# ws_designs

A curated collection of 167 web design aesthetics — each with a detailed design guide and a standalone HTML/CSS example page.

## What's Inside

Every aesthetic includes:

- **Design guide** (`designs/*.md`) — color palettes, typography, layout principles, CSS techniques, and visual references
- **Working example** (`examples/*.html`) — a self-contained HTML page demonstrating the aesthetic in practice

Built with pure HTML5/CSS3, Google Fonts, and vanilla JS. No build tools or frameworks required.
